# PIC_Microcontroller_Projects

**Project Status:** Active

This repository contains a collection of personal experiment projects focused on learning register-level programming with PIC microcontrollers using MPLAB X. 

Each `.c` file is a self-contained lab exercise designed to run on either a breadboard setup or the *Low Pin Count Demo Board*. 

### Supported Hardware
* **PIC16F690 (DIP-20):** The primary MCU used for all current exercises.

### Project Structure
* **Documentation:** The header at the top of each individual C file specifies the exact functionality, configuration, system frequency, and required hardware components.
* **Source Files:** Only the standalone `.c` source files are provided.
